Steps to execute:
1) make.
2) ./simulator.
3) Enter choice "1" to load the file.
4) Enter the file Name which has instruction sequence. No space between the file name. e.g. - input.txt.
5) Enter choice "2" to initialise.
6) Enter choice "3" to simulate for all cycles or "4" to simulate for n cycles.
7) if Entered "4" then enter the number of cycle.
8) Enter choice "5" to display.
9) Enter choice "6" to Exit or Enter "1" to load a new file.
